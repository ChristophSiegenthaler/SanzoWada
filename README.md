# Sanzo Wada — Color Combination Explorer

An interactive color palette tool inspired by *A Dictionary of Color Combinations* (1933) by Sanzo Wada.

## What it does

- Pick a base color from an HSL color wheel
- Generates harmonious palettes using standard color theory modes (triadic, analogous, complementary, etc.) plus a "Wada muted" mode inspired by traditional Japanese dyeing
- Click any hex code to copy it
- Browse and apply Wada-inspired palette variations

## Usage

No dependencies. Open `index.html` in a browser.

Or view it live: [GitHub Pages link]

## Controls

| Control | Description |
|---|---|
| Color wheel | Click or drag to set base color |
| Number of colors | 2 – 5 colors in the palette |
| Harmony mode | Analogous, triadic, complementary, split, tetradic, Wada muted |
| Variation cards | Click to apply a preset variation |

## Credit

Inspired by Sanzo Wada's original work, built around observation of nature and traditional Japanese dyeing — dusty, aged, seasonal tones over pure primaries.
